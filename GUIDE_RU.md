# Инструкция для начинающих — AudioShield

Этот документ написан для людей, которые никогда не занимались обработкой звука и не работали с командной строкой. Всё объяснено максимально просто, шаг за шагом.

---

## Что делает эта программа?

Представьте, что у вас есть аудиофайл — например, музыкальная дорожка или запись голоса. Существуют автоматические системы, которые «слушают» такие файлы и узнают их по звуковому отпечатку — как Shazam, только для других задач.

AudioShield изменяет файл так, чтобы такие системы его не узнали. При этом на слух изменения могут быть совершенно незаметны (при мягких настройках) или незначительны. Программа не портит файл — она тонко меняет его цифровую структуру.

---

## Шаг 1. Установить Python

Python — это язык, на котором написана программа. Без него она не запустится.

1. Перейдите на сайт https://www.python.org/downloads/
2. Нажмите большую жёлтую кнопку «Download Python 3.x.x»
3. Запустите скачанный файл
4. **ОЧЕНЬ ВАЖНО:** в самом начале установки поставьте галочку **«Add Python to PATH»** (она внизу окна). Без этого ничего не заработает.
5. Нажмите «Install Now» и дождитесь окончания

Проверка: откройте меню Пуск, найдите «cmd» или «Командная строка», запустите и напечатайте:
```
python --version
```
Должно появиться что-то вроде `Python 3.12.3`. Если появилось — всё хорошо.

---

## Шаг 2. Установить FFmpeg

FFmpeg — это отдельная программа, которая занимается самой обработкой звука. AudioShield использует её «под капотом».

1. Перейдите на https://ffmpeg.org/download.html
2. В разделе Windows найдите ссылку на сборку от **gyan.dev** или **BtbN**
3. Скачайте архив с пометкой `ffmpeg-release-essentials.zip` (или похожее название)
4. Распакуйте архив. Внутри будет папка типа `ffmpeg-7.0-essentials_build`
5. Переименуйте эту папку в просто `ffmpeg` и переместите её в корень диска C: — должно получиться `C:\ffmpeg`
6. Теперь нужно добавить FFmpeg в PATH:
   - Нажмите правой кнопкой на «Этот компьютер» → «Свойства»
   - «Дополнительные параметры системы» → «Переменные среды»
   - В нижнем списке («Системные переменные») найдите строку **Path** и дважды кликните по ней
   - Нажмите «Создать» и введите: `C:\ffmpeg\bin`
   - Нажмите OK везде

Проверка: откройте новое окно командной строки (важно — новое, а не старое) и введите:
```
ffmpeg -version
```
Должна появиться информация о версии FFmpeg. Если появилась — отлично.

---

## Шаг 3. Установить библиотеки Python

Откройте командную строку и введите одну команду:

```
pip install PyQt6 mutagen pyqtgraph numpy
```

Подождите, пока всё установится. Если появятся ошибки с текстом `error` — скопируйте текст ошибки и обратитесь за помощью.

---

## Шаг 4. Запустить программу

1. Откройте папку с программой (там должен быть файл `gui_qt.py`)
2. Откройте командную строку в этой папке:
   - Зажмите Shift и кликните правой кнопкой мыши на пустом месте в папке
   - Выберите «Открыть окно PowerShell здесь» или «Открыть командную строку здесь»
3. Введите:
   ```
   python gui_qt.py
   ```
4. Откроется окно программы

**Совет:** создайте файл `run.bat` в папке с программой со следующим содержимым:
```
python gui_qt.py
pause
```
Тогда можно будет просто дважды кликнуть по нему, чтобы запустить программу.

---

## Шаг 5. Интерфейс программы — что где

Когда программа открылась, вы увидите несколько зон:

- **Левая панель** — список файлов для обработки. Сюда перетаскиваются или добавляются файлы.
- **Центральная область** — вкладки с настройками методов обработки.
- **Нижняя часть** — кнопки запуска, прогресс-бар.
- **Правая панель** (если есть) — лог: здесь отображается, что происходит с каждым файлом.
- **Форма волны** (если есть) — визуализация звука выбранного файла.

---

## Шаг 6. Добавить файлы

Есть два способа:
- **Перетащить** файлы мышью прямо в левую панель
- Нажать кнопку **«Добавить файлы»** (или `Ctrl+O`) и выбрать файлы через диалог

Поддерживаются: MP3, FLAC, WAV, OGG, M4A, AAC и другие.

---

## Шаг 7. Выбрать метод обработки

Посмотрите на вкладки в центре:

- **Основные** — классические эффекты (реверберация, хорус и т.д.)
- **Частотные** — работа с частотами звука (эквалайзер, фильтры)
- **Шумы** — добавление различных типов шума
- **Текстуры** — специальные методы (Спектральный джитер, VK Инфразвук)

Для начала рекомендуем попробовать **Спектральный джитер** — это наиболее универсальный метод:

1. Перейдите на вкладку «Текстуры»
2. Найдите раздел «Спектральный джитер»
3. Поставьте галочку, чтобы включить метод
4. Нажмите кнопку **«Мягкий»** — это самый безопасный пресет, почти не меняющий звук
5. Готово — метод настроен

---

## Шаг 8. Настроить выходную папку и имя файла

Найдите поле с шаблоном имени файла. По умолчанию стоит что-то вроде `{name}_out.{ext}` — это значит, что обработанный файл будет называться так же, как исходный, но с добавлением `_out` в конце.

Вы также можете выбрать папку для сохранения результата.

---

## Шаг 9. Запустить обработку

1. Убедитесь, что файлы добавлены и хотя бы один метод включён
2. Нажмите кнопку **«Обработать»** (или `F5`)
3. Следите за прогресс-баром и логом
4. Когда обработка закончится, в логе появится сообщение об успехе

Результаты будут в указанной папке.

---

## Пресеты — сохранение и загрузка настроек

Если вы нашли хорошую комбинацию настроек, её можно сохранить как пресет:
- `Ctrl+S` — сохранить текущие настройки в файл
- `Ctrl+L` — загрузить ранее сохранённый пресет

Это удобно, когда нужно применять одинаковую обработку к разным наборам файлов.

---

## Конвертер

В программе есть встроенный конвертер форматов. Он не занимается защитой — только меняет формат файла (например, из FLAC в MP3). Найдите его на соответствующей вкладке, добавьте файлы, выберите формат и нажмите «Конвертировать».

## Частые проблемы

**«python» не является командой / не распознаётся**
→ Не была поставлена галочка «Add Python to PATH» при установке. Переустановите Python, не забыв про галочку.

**ModuleNotFoundError: No module named 'PyQt6'**
→ Библиотеки не установлены. Выполните: `pip install PyQt6 mutagen pyqtgraph numpy`

**FFmpeg not found / ffmpeg не найден**
→ FFmpeg не добавлен в PATH. Проверьте шаг 2. Убедитесь, что открываете новое окно командной строки после изменения PATH.

**Форма волны не отображается**
→ Не установлены pyqtgraph или numpy. Выполните: `pip install pyqtgraph numpy`. Это не влияет на работу обработки.

**Обработка завершается с ошибкой**
→ Прочитайте лог — там будет описание ошибки. Чаще всего причина в некорректном файле или в том, что FFmpeg не найден.

**Файл после обработки звучит плохо**
→ Выберите более мягкий пресет (например, «Мягкий» вместо «Сильный») или уменьшите параметры вручную.

2. Включите только «Спектральный джитер» с пресетом «Мягкий»
3. Запустите обработку
4. Сравните исходный и обработанный файл на слух

Если разница незаметна — программа работает именно так, как задумано.
