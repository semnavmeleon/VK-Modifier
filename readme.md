# VK Modifier

Инструмент для изменения цифрового отпечатка аудиофайлов с графическим интерфейсом. Позволяет модифицировать MP3-файлы незаметно для слуха и конвертировать аудио между 26 форматами.

---

## Возможности

- 25+ методов обработки аудио

- Живой предпросмотр waveform до и после

- Пакетная обработка в несколько потоков

- Конвертация между 26 аудиоформатами

- Drag & Drop файлов

- Сохранение и загрузка пресетов

- Редактирование метаданных и обложек

- Гибкий шаблон имён выходных файлов

---

## Требования

Python 3.8+

FFmpeg в системном PATH или рядом с исполняемым файлом

Зависимости из requirements.txt

**Python-зависимости:**

Скачайте с [ffmpeg.org](https://ffmpeg.org/) и добавьте в PATH. Либо положите ffmpeg.exe / ffmpeg в папку с программой.

Проверка: ffmpeg -version

## Режимы работы

Приложение имеет два режима, переключение — кнопками в верхней панели.

### Модификатор

Изменение цифрового отпечатка MP3-файлов без слышимых изменений. Добавьте MP3-файлы, выберите методы обработки на вкладках, настройте метаданные и параметры вывода, нажмите «Запустить обработку».

### Конвертер

Конвертация между 26 аудиоформатами. Добавьте файлы любых поддерживаемых форматов, выберите выходной формат и качество из выпадающих списков, нажмите «Запустить конвертацию». Формат входного файла определяется автоматически, указывать его не нужно.

**Поддерживаемые форматы:**

Lossy: MP3, AAC, M4A, OGG Vorbis, Opus, WMA, AC3, DTS, MP2, Musepack, Speex, AMR

Lossless: FLAC, WAV, AIFF, ALAC, WavPack, Monkey's Audio, True Audio, Shorten, Ogg FLAC

Без сжатия/контейнеры: AU, Matroska Audio, Core Audio Format

Конвертация работает во всех направлениях. Для lossy-форматов настраивается битрейт или уровень качества.

## Методы обработки

### Базовые

Pitch Shift — сдвиг тональности на ±5 полутонов  
Time Stretch — изменение скорости 0.90–1.10x  
EQ — эквализация (стандарт, mid-cut, air)  
Silent Pad — добавление тишины в конец

### Спектральные

Phase Invert — инверсия фазы правого канала  
Phase Scramble — синусоидальная модуляция фазы  
DC Shift — постоянное смещение сигнала  
Resample Drift — дрейф частоты дискретизации  
Haas Delay — стерео-задержка 0–50 мс  
Ultrasonic Noise — подмешивание ультразвука  
Dither Attack — шум квантования  
ID3 Padding Attack — мусорные данные в тегах

### Текстурные

Spectral Masking — точечное подавление частотных пиков  
Concert Emulation — эмуляция концертной записи  
Mid/Side Processing — раздельная обработка центра и боков  
Psychoacoustic Noise — высокочастотный интерференционный шум  
Saturation — аналоговое насыщение  
Temporal Jitter — микровариации скорости  
Spectral Jitter — случайные провалы в спектре  
VK Инфразвук — инфразвуковая модуляция 10–20 Гц

### Дополнительные

Trim — обрезка начала трека  
Cut Fragment — вырезание фрагмента  
Fade Out — плавное затухание  
Merge — сращивание двух треков  
Broken Duration — подмена длительности в метаданных

### Технические

Bitrate Jitter — рандомизация битрейта  
Frame Shift — удаление Xing-заголовка  
Fake Metadata — случайный мусор в comment  
Reorder Tags — переупорядочивание ID3 тегов

## Пресеты

Настройки методов обработки можно сохранять, загружать, экспортировать и импортировать в JSON. Пресеты хранятся в файле vk_modifier_config.json рядом с программой.

---

## Горячие клавиши

Ctrl+O — добавить файлы  
Ctrl+A — выделить все файлы в списке  
Delete — удалить выбранный файл  
Ctrl+C — копировать имена выделенных файлов  
Ctrl+S — сохранить пресет

Для текстовых полей работают стандартные Ctrl+C, Ctrl+V, Ctrl+X, Ctrl+A.

---

## Drag & Drop

При установленном tkinterdnd2 доступно перетаскивание аудиофайлов на список файлов или поле extra-трека, а также изображений на поле обложки.

---

## Шаблон имени файла

Доступные переменные: {n}, {original}, {title}, {artist}, {album}, {year}

Примеры:

VK_{n:03d}_custom -> VK_001_custom.mp3

{artist} - {title} -> Исполнитель - Название.mp3

modified_{original} -> modified_original_name.mp3
