**VK Modifier**
Программа для изменения цифрового отпечатка MP3-файлов. Позволяет обходить системы автоматического распознавания аудио, при этом изменения не слышны человеческому уху.

**⚠️ Требуется наличие FFmpeg в системной PATH или просто положите его рядом с исполняемым файлом чтобы он мог вызываться.**

**ВОЗМОЖНОСТИ**
- 25+ способов обработки звука
- Живой предпросмотр волны до и после
- Обработка нескольких файлов сразу
- Сохранение и загрузка пресетов
- Перетаскивание файлов мышкой (Drag & Drop)

**ОТКАЗ ОТ ОТВЕТСТВЕННОСТИ**
Обрабатывай только аудио файлы, которые принадлежат ВАМ. Если вы решите с помощью этого скрипта обходить авторские права, блокировки или системы распознавания - это ваш выбор и ваша ответственность.
