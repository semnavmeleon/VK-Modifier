# AudioShield — Инструмент защиты аудиофайлов

Программа для пакетной обработки аудиофайлов с целью противодействия несанкционированному распознаванию и снятию отпечатков. Поддерживает 30+ методов обработки, работает через FFmpeg, интерфейс на PyQt6.

---

## Возможности

- Пакетная обработка: MP3, FLAC, WAV, OGG, M4A, AAC и другие форматы
- 30+ методов обработки, разбитых по вкладкам
- Настраиваемые пресеты (сохранение/загрузка конфигурации)
- Встроенный конвертер форматов
- Отображение формы волны (требует pyqtgraph + numpy)
- Лог обработки с цветовой индикацией
- Поддержка ID3/Vorbis метаданных через mutagen
- Шаблоны имён выходных файлов
- Горячие клавиши

## FFmpeg

Программа требует `ffmpeg` и `ffprobe` в системном PATH.

**Windows:**
1. Скачать с https://ffmpeg.org/download.html (сборка BtbN или gyan.dev)
2. Распаковать архив, например в `C:\ffmpeg`
3. Добавить `C:\ffmpeg\bin` в переменную PATH (Система → Дополнительно → Переменные среды)
4. Проверить: `ffmpeg -version` в cmd

**Linux:**
```
sudo apt install ffmpeg        # Debian/Ubuntu
sudo pacman -S ffmpeg          # Arch
```

**macOS:**
```
brew install ffmpeg
```

## Методы обработки

### Вкладка «Основные»

| Метод | Описание |
|-------|----------|
| Нормализация | Выравнивание громкости (LUFS / пиковая / EBU R128) |
| Реверберация | Добавление помещения через FFmpeg `aecho` |
| Задержка (Delay) | Классическое эхо с настройкой времени и обратной связи |
| Хорус | Модуляционный эффект (`chorus`) |
| Дисторшн | Клиппирование сигнала (`acrusher`) |
| Флэнжер | Флэнжер через `flanger` |
| Фейзер | Фазовый сдвиг через `aphaser` |
| Тремоло | Амплитудная модуляция (`tremolo`) |
| Вибрато | Частотная модуляция (`vibrato`) |

### Вкладка «Частотные»

| Метод | Описание |
|-------|----------|
| Эквалайзер | Полки/пики, до 8 полос |
| Фильтр высоких | High-pass (HPF) |
| Фильтр низких | Low-pass (LPF) |
| Notch-фильтр | Режекторный фильтр на заданной частоте |
| Pitch Shift | Сдвиг высоты тона без изменения темпа |
| Time Stretch | Растяжение/сжатие времени без изменения тона |

### Вкладка «Шумы»

| Метод | Описание |
|-------|----------|
| Белый шум | Добавление белого шума заданной амплитуды |
| Розовый шум | Розовый шум (спад −3 дБ/окт) |
| Случайные паузы | Вставка случайных тихих участков |
| Битовый краш | Снижение разрядности (bit crush) |
| Ресэмплинг | Снижение и восстановление частоты дискретизации |

### Вкладка «Текстуры» — Спектральный джитер

Инжекция notch-фильтров для нарушения спектральной сигнатуры.

**Режимы:**
- Случайный — случайные полосы при каждом запуске
- Конструктор — ручная настройка полос (частота, глубина дБ, ширина Q)

**Встроенные пресеты конструктора:**

| Пресет | Полос | Макс. глубина | Описание |
|--------|-------|---------------|----------|
| Мягкий | 2 | 3 дБ | Практически незаметно на слух |
| Лёгкий | 3 | 6 дБ | Едва слышимые изменения |
| Средний | 5 | 9 дБ | Заметная обработка |
| Сильный | 6 | 15 дБ | Выраженное изменение тембра |
| Жёсткий | 8 | 22 дБ | Сильное искажение спектра |
| Максимум | 11 | 30 дБ | Агрессивное подавление |

### Вкладка «Текстуры» — VK Инфразвук

Внедрение сверхнизкочастотного сигнала (1–20 Гц) под порог слышимости.

**Формы волны:**

| Режим | Описание |
|-------|----------|
| Синус | Чистая синусоида |
| Меандр | Прямоугольная волна |
| Пила | Пилообразная волна |
| Гармоники | Основной тон + обертоны |
| Максимум | Комбинация форм с фазовым сдвигом |

Стерео-генерация: L-канал на фазе 0°, R-канал на фазе ≈30° (0.5236 рад). Сигнал сохраняется при mono-даунмиксе.

## Шаблоны имён файлов

| Переменная | Подставляется |
|------------|---------------|
| `{name}` | Имя исходного файла без расширения |
| `{ext}` | Расширение исходного файла |
| `{date}` | Дата обработки `YYYYMMDD` |
| `{time}` | Время обработки `HHMMSS` |
| `{n}` | Порядковый номер в пакете |
| `{method}` | Краткое название применённых методов |

Пример: `{name}_out_{date}.{ext}` → `mysong_out_20260516.mp3`

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+O` | Добавить файлы |
| `Ctrl+D` | Удалить выбранные из списка |
| `Ctrl+A` | Выбрать все файлы |
| `Ctrl+S` | Сохранить пресет |
| `Ctrl+L` | Загрузить пресет |
| `F5` / `Ctrl+R` | Запустить обработку |
| `Esc` | Остановить обработку |
| `Ctrl+Q` | Выход |
